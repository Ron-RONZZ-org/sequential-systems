# sequential systems

This repository host code related to manipulation of measurement instruments, notamment those listed below:

- National Instruments (NI)
    - protek 506
    - NI-DAQmx compatible systems

This repository does not pretend or even aim to be comprehensive. The instruments are simply those present at Université de Lorraine (Metz).

## Dev

```bash
git clone https://github.com/Ron-RONZZ-org/sequential-systems.git
poetry install
poetry env activate
```

For more help on python and poetry package manager, see [this cheatsheet](https://ronzz.org/py/).
